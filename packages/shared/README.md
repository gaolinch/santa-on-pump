# @santa/shared

Shared types and utilities for the Santa monorepo.

## Usage

Import shared types in your apps:

```typescript
import { GiftSpec, Winner, DayStats } from '@santa/shared';
```

## Contents

- **Types** - TypeScript interfaces for API contracts, gifts, transactions
- **Constants** - Season dates, allocation percentages, etc.
- **Utilities** - Shared helper functions

## Building

```bash
npm run build
```

This package is automatically built when you run `npm run build` from the root.
